Project Overview

Seeku - Complete Health Calculators is a versatile health calculator app built with Flutter and Dart. This app allows users to calculate their Body Mass Index (BMI), track breath counts, determine caloric needs, manage hydration, monitor heart rates, and gain valuable health insights. This documentation will guide you through the installation, usage, and deployment of the app.

  • 12+ Advanced Health Calculators:
    • BMI Calculator

    • Caloric Needs Calculator
    • Heart Rate Calculator
    • Pregnancy Due Date Calculator
    • Ovulation Period Calculator
    • Blood Volume Calculator
    • Blood Donation Eligibility Calculator
    • Blood Alcohol Content (BAC) Calculator

Setup

Installation

Prerequisites:

  • Ensure you have Flutter SDK installed on your machine. Follow the installation guide on the Flutter website.
  • Install Dart SDK, which comes bundled with Flutter.
  • Set up an IDE (e.g., Android Studio, Visual Studio Code) with Flutter and Dart plugins.

Install Dependencies:

flutter pub get

Using the App

Launch the App:

flutter run

Calculate BMI:

Enter your weight, height, age, and gender in the respective fields, then tap the "Calculate" button to view your BMI and health insights.

Track Breaths:

Set the duration and tap the "Start Counting" button to track your breath count. The app will display your results once the duration ends.

Calculate Caloric Needs:

Input your activity level, age, weight, and height to find out your daily caloric needs for effective diet management.

Calculate Water Intake:

Use the water intake calculator to determine how much water you should drink daily based on your weight and activity level.

Monitor Heart Rate:

Input your resting heart rate to calculate your target heart rate zones for different exercise intensities.

Change Language:

Navigate to the settings and select your preferred language from the available options.

Getting the APK Release

Build the APK:

flutter build apk --release

Locate the APK:

After the build is complete, you can find the APK file in the following directory: 

build/app/outputs/flutter-apk/app-release.apk

Install the APK:

You can install the APK on any Android device by transferring the file and opening it, or by using ADB: 

adb install build/app/outputs/flutter-apk/app-release.apk

Building for iOS

Set Up iOS Development Environment: Ensure you have Xcode installed on your Mac.

Build the iOS App:

flutter build ios --release

Locate the iOS Build:

The iOS build files will be located in the ios/ directory of your project. You can open the .xcworkspace file in Xcode to run the app on a simulator or a physical device.

Integrating AdMob

Create an AdMob Account: Sign up for an AdMob account at AdMob.

Create an Ad Unit: In your AdMob dashboard, create a new ad unit and note down the Ad Unit ID.

Update AndroidManifest.xml: To integrate AdMob, you need to add the following permissions and metadata to your AndroidManifest.xml file located at android/app/src/main/AndroidManifest.xml: 

<manifest xmlns:android="http://schemas.android.com/apk/res/android" package="com.example.seeku_health_calc">
    <application>
        <!-- Other configurations -->

        <!-- AdMob App ID -->
        <meta-data
            android:name="com.google.android.gms.ads.APPLICATION_ID"
            android:value="YOUR_ADMOB_APP_ID"/>

        <!-- Other configurations -->
    </application>

    <uses-permission android:name="android.permission.INTERNET"/>
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>
</manifest>

Replace YOUR_ADMOB_APP_ID with your actual AdMob App ID.

Load and Display Ads: Use the following code snippet to load and display ads in your app:

Create a .env file in the root of your project with the following configuration: 

# AdMob Configuration
ADMOB_APP_ID=""
ADMOB_BANNER_AD_UNIT_ID_ANDROID=""
ADMOB_BANNER_AD_UNIT_ID_IOS=""
ADMOB_INTERSTITIAL_AD_UNIT_ID_ANDROID=""
ADMOB_INTERSTITIAL_AD_UNIT_ID_IOS=""

flutter run

flutter build apk --release
